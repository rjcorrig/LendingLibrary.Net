# Lending Library

Lending Library is a fun, free website to meet other bibliophiles online, browse their bookshelves and trade books!

## System Requirements

* .NET Framework 4.5.2
* ASP.NET MVC 5.2.3
* Entity Framework 6.1.3
* MariaDB 10.2
